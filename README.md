# BarberShop Appointment (iOS)

An iOS app for a barber shop that lets users sign up / sign in and book appointments.

## Features

- Firebase Authentication (email/password)
- Appointment booking stored in Firestore
- Home screen with map/location UI

## Requirements

- Xcode (latest stable recommended)
- iOS deployment target: 18.5+
- A Firebase project (Firestore + Auth enabled)

## Running Locally

1. Open `AppoitmentCalendar.xcodeproj` in Xcode.
2. Ensure `GoogleService-Info.plist` matches your Firebase project.
3. Select a simulator/device and Run.

## Repo Notes

- `AppoitmentCalendar.xcodeproj/xcuserdata/*` is intentionally ignored (user-specific Xcode state).
